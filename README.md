# ordinario-ftw
# Biblioteca Digital - Proyecto Final FTW

Proyecto desarrollado para la asignatura de **Fundamentos de Tecnologías Web (NRC 40260)**.

## Descripción
Sitio web interactivo para la gestión y visualización de un catálogo bibliográfico. El sitio permite el registro de usuarios, filtrado dinámico de libros y una interfaz responsiva.

## Tecnologías Utilizadas
- **HTML5:** Estructura semántica (header, nav, main, footer).
- **CSS3:** Estilos responsivos utilizando *Media Queries*.
- **JavaScript:** Lógica de componentes, persistencia con `localStorage` y manipulación del DOM.
- **XML:** Gestión de datos del catálogo mediante archivos estructurados.

## Funcionalidades
1. **Registro de Usuario:** Validación y persistencia de sesión.
2. **Catálogo Dinámico:** Carga de libros desde archivos XML.
3. **Diseño Responsivo:** Adaptación automática a dispositivos móviles y escritorio.

## Uso de Inteligencia Artificial (Prompts)
Para el desarrollo de este proyecto, se utilizó asistencia de IA en las siguientes áreas:

**Lógica y Programación:**
* *Prompt:* "¿Cómo puedo persistir el nombre de usuario usando localStorage en JavaScript?"
* *Prompt:* "¿Cómo crear una función de filtrado para mostrar datos de un XML en una tabla?"

**Diseño y Experiencia de Usuario (UX/UI):**

* *Prompt:* "¿Cómo implementar un diseño responsivo utilizando Media Queries para que el catálogo se adapte a dispositivos móviles?"
* *Prompt:* "¿Qué estructura de jerarquía visual es recomendable para una biblioteca digital?"

## Estructura del Repositorio
- `/css`: Hojas de estilos.
- `/html`: Archivos principales del sitio.
- `/javascript`: Lógica del sitio.
- `/data`: Archivos XML del catálogo.
